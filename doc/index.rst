.. module:: gh_bugs

:mod:`gh_bugs`
==============

:mod:`gh_bugs` is a very simple client for working with `GitHub's issue
tracker`_ from the command line.

It is written in Python_, and requires v2.6 or later.  :mod:`versionah` is
released under the `GPL v3`_
